## وب‌سایت چیست؟

مجموعه‌ای از فایل‌ها (HTML, CSS, JS، عکس‌ها و…) که روی یک سرور قرار دارند و مرورگر آن‌ها را نمایش می‌دهد.

## مرورگر چیست؟

برنامه‌ای مثل Chrome یا Firefox که فایل‌های وب (HTML/CSS/JS) را **تفسیر** می‌کند و آن‌ها را به شکل یک صفحه قابل دیدن نمایش می‌دهد.

## HTML / CSS / JavaScript چه کاری می‌کنند؟

- **HTML**: ساختار و اسکلت صفحه را می‌سازد. (عنوان، متن، عکس، جدول، فرم)
- **CSS**: ظاهر و طراحی صفحه را مشخص می‌کند. (رنگ، فونت، اندازه، فاصله، چیدمان)
- **JavaScript**: رفتار و منطق صفحه را فعال می‌کند. (کلیک، انیمیشن، محاسبه، ارتباط با سرور)

## ساخت پروژه در یک فولدر

یک پوشه بساز و تمام فایل‌های مربوط به سایت (HTML, CSS, JS, تصاویر) را داخل همان پوشه قرار بده تا مرورگر بتواند همه را درست پیدا کند.

## ساخت فایل `.html`

داخل پروژه یک فایل بساز با پسوند:

```
index.html
```

مرورگر این فایل را به عنوان صفحه اصلی می‌شناسد.

# تگ‌های متنی

## h1 → h6

تگ‌های عنوان هستند.  
h1 بزرگترین و مهم‌ترین عنوان، h6 کوچک‌ترین عنوان است.

مثال:

```html
<h1>عنوان اصلی</h1>
```

## p

تگ پاراگراف؛ برای نمایش متن‌های چند‌خطی و جملات.

```html
<p>این یک پاراگراف است.</p>
```

## br

خط جدید ایجاد می‌کند؛ متن را به خط بعد می‌برد. تگ بسته ندارد.

```html
سلام<br />خوبی؟
```

## span

برای دور یک بخش کوچک از متن استفاده می‌شود تا روی آن استایل بگذاریم یا جداگانه هدف بگیریم.

```html
<p>این یک <span>کلمه مهم</span> است.</p>
```
